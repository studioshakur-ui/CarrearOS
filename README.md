# Career Agent

Minimal foundation for a premium AI Career Copilot focused on Gulf hiring.

## Structure

- `apps/web` - Next.js App Router frontend
- `packages/core` - shared domain scaffolding
- `packages/ai` - future AI orchestration scaffolding
- `packages/jobs` - future job ingestion scaffolding
- `supabase` - migrations and platform setup
- `docs` - lightweight project docs

## Status

This repository currently contains foundation scaffolding only.
